
# GitHub Copilot Instructions – AlmacenGestion

This repository contains a Python inventory/store management system.

Coding guidelines for Copilot suggestions:

- Language: Python 3
- Framework: lightweight Flask-style application
- Templates: Jinja HTML
- Database handled in database.py

Rules:

1. Maintain existing naming conventions in Spanish.
2. Prefer simple Python standard library solutions.
3. Avoid heavy external dependencies.
4. Ensure compatibility with existing modules.

Preferred patterns:

- Clear functions
- Simple SQL interactions
- Readable logic over clever code

If generating new features:
- Follow structure used in app.py
- Keep UI changes inside templates/
