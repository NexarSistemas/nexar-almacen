🚀 Inicio Rápido

Bienvenido al Sistema de Gestión para Almacenes. Esta guía te ayudará a configurar y usar el sistema desde el primer día.

Primeros pasos recomendados
1
Configuración del negocio: Andá a Sistema → Configuración y completá el nombre, dirección, CUIT y datos de tu almacén. Estos datos aparecerán en los tickets.
2
Cargá tus productos: En Productos → Catálogo, hacé clic en «Nuevo Producto» e ingresá cada artículo con su código, precio y costo.
3
Ajustá el stock inicial: En Productos → Stock, editá cada producto y colocá la cantidad actual que tenés en el depósito.
4
Configurá usuarios: Creá un usuario para cada vendedor (rol «Usuario») para que puedan acceder con sus propias credenciales.
5
Empezá a vender: ¡Ya podés usar el Punto de Venta! Cada venta actualizará el stock automáticamente.
💡 Tip: Los datos de demo (25 productos de ejemplo, clientes y ventas ficticias) están cargados para que puedas explorar el sistema antes de ingresar tus datos reales. Podés borrarlos desde el catálogo de productos.

  • Vendedor: vendedor / vendedor123
⚠️ Importante: Cambiá las contraseñas por defecto inmediatamente desde Sistema → Usuarios.
Gestión de usuarios (solo admin)

Desde Sistema → Usuarios podés crear nuevos vendedores, cambiar contraseñas, activar/desactivar y cambiar roles. Un administrador no puede desactivar su propia cuenta.

🏠 Dashboard

El panel principal muestra un resumen en tiempo real de tu negocio:

  • Ventas de hoy: Total y cantidad de tickets del día actual.
  • Ventas del mes: Acumulado mensual.
  • Gastos del mes: Total de gastos registrados este mes.
  • Alertas de stock: Productos sin stock o en nivel crítico.
  • Gráfico mensual: Evolución de ventas en el año.
  • Facturas por vencer: Las 5 más próximas a vencer con proveedores.

📦 Catálogo de Productos

Agregar un producto nuevo
1
Clic en «Nuevo Producto» (botón azul, arriba a la derecha).
2
El sistema genera automáticamente un código interno (PRD-XXXX). Podés cambiar el código de barras.
3
Completá: descripción, categoría, unidad, costo y precio de venta.
4
El margen de ganancia se calcula automáticamente: verde ≥35%, amarillo 20-34%, rojo <20%.
5
Indicá el stock inicial, mínimo y máximo. El mínimo dispara alertas cuando el stock baja.
Productos por peso (fiambres, quesos, etc.)

Activá la opción «Venta por peso/kg». En el Punto de Venta pedirá el peso en lugar de la cantidad.

Exportación del catálogo

Desde el listado de productos podés exportar a:

  • Excel (.xlsx): Planilla con colores por estado de stock, filtros automáticos y formato de moneda.
  • PDF imprimible: Tabla lista para imprimir con todos los precios y estados.
💡 La exportación solo está disponible para Administradores.

📊 Control de Stock

El stock se actualiza automáticamente con cada venta y compra registrada. Los niveles de alerta son:

EstadoCondiciónAcción recomendada
SIN STOCKStock ≤ 0Urgente: pedir reposición
CRÍTICOStock ≤ stock mínimoPedir pedido inmediato
BAJOStock ≤ 150% del mínimoPlanificar reposición
NORMALEntre mínimo y máximoSin acción
EXCESOStock ≥ máximoReducir compras

Hacé clic en cualquier producto para editar stock actual, mínimo, máximo y proveedor habitual directamente desde la pantalla.

🛒 Punto de Venta

Cómo realizar una venta
1
Buscá el producto: Escaneá el código de barras con la pistola lectora (presionando Enter), o escribí el código interno o la descripción para búsqueda instantánea.
2
Confirmá la cantidad: Ajustá la cantidad (o el peso si es producto por kg) y el descuento individual (%).
3
Agregá más ítems: Repetí para cada producto. Podés agregar tantos como necesites.
4
Seleccioná el cliente (opcional): Si es cuenta corriente, es obligatorio seleccionar el cliente.
5
Elegí el medio de pago: Efectivo, Débito, Crédito, Transferencia, QR o Cuenta Corriente.
6
Confirmá la venta: El sistema genera el ticket, actualiza el stock y registra en caja.
Búsqueda instantánea de productos

Al tipear más de 2 letras en el buscador, aparece un dropdown con sugerencias en tiempo real. Los códigos de barras numéricos se procesan con Enter.

💡 Lector de barras: Conectá la pistola USB. Al escanear, el código se ingresa automáticamente. Si el artículo está en el sistema, se agrega al carrito con Enter.
⚠️ En modo demo, el sistema permite hasta 30 ventas. Al alcanzar el límite, bloqueará nuevas ventas hasta activar la licencia completa.

📝 Historial de Ventas

Muestra todas las ventas registradas. Podés filtrar por:

  • Texto: Busca en nombre del cliente, número de ticket y productos.
  • Rango de fechas: Desde / hasta una fecha determinada.

Hacé clic en cualquier venta para ver el detalle completo e imprimir el ticket.

🛍️ Compras

Registrá cada ingreso de mercadería. Al registrar una compra:

  • El stock del producto aumenta automáticamente.
  • Si especificás un costo unitario, actualiza el costo del producto.
  • Queda registrada la fecha, el remito/factura y el proveedor.

💰 Caja Diaria

Apertura de caja

Al inicio del día ingresá el saldo de apertura (efectivo disponible) y el responsable.

Durante el día

Las ventas en efectivo se acumulan automáticamente. Podés registrar gastos del día (alquiler, servicios, etc.) directamente desde caja.

Cierre de caja

Al finalizar el día ingresá el saldo real (efectivo contado). El sistema calcula la diferencia:

  • Diferencia positiva: Sobra efectivo (posible venta no registrada o error de cambio).
  • Diferencia negativa: Falta efectivo (posible error o gasto no registrado).

💸 Gastos

Registrá todos los egresos del negocio. Cada gasto se clasifica como:

  • SI (necesario): Gastos inevitables (alquiler, luz, impuestos).
  • NO (prescindible): Gastos que podrían eliminarse para mejorar la rentabilidad.
  • A REVISAR: Gastos que requieren análisis.

Esta clasificación aparece en el módulo de Análisis y permite identificar oportunidades de ahorro.

👥 Cuentas Corrientes — Clientes

Gestioná la deuda de tus clientes que compran a crédito.

  • Al registrar una venta en «Cuenta Corriente», se crea automáticamente un movimiento de debe.
  • Podés registrar pagos parciales o totales desde el detalle del cliente.
  • El sistema alerta cuando hay vencimientos próximos.
  • Cada cliente tiene un límite de crédito configurable.

🏭 Cuentas Corrientes — Proveedores

Controlá las deudas con tus proveedores.

  • Registrá las facturas de compra con fecha de vencimiento.
  • El sistema te alerta 30 días antes del vencimiento.
  • Registrá pagos parciales o totales por factura.
  • Las facturas vencidas se resaltan en rojo.

📈 Estadísticas

Gráficos de ventas para analizar el rendimiento:

  • Ventas mensuales: Evolución a lo largo del año (podés seleccionar el año).
  • Ventas semanales: Últimas 8 semanas.
  • Medios de pago: Distribución del mes actual.
  • Temporadas: Comparativa Verano / Otoño / Invierno / Primavera.
  • Por categoría: Qué rubros generan más ventas.

🔍 Análisis de Rentabilidad

  • Rentabilidad del mes: Ventas, gastos, utilidad y % de rentabilidad.
  • Top productos: Los más vendidos por unidades y por monto.
  • Menos vendidos: Productos que podrían descontinuarse.
  • Gastos por categoría: Con clasificación necesario/prescindible.
  • Recomendaciones estacionales: Qué productos priorizar según la temporada.

⚙️ Configuración (solo Administrador)

  • Datos del negocio: Nombre, dirección, CUIT, teléfono — aparecen en los tickets.
  • Márgenes: Porcentajes mínimo y objetivo para las alertas de margen en productos.
  • Días de alerta: Con cuántos días de anticipación alertar sobre vencimientos.
  • Categorías: Agregá o eliminá categorías de productos.

🔑 Licencia

El sistema incluye un modo demo con límites de prueba (30 ventas, 100 productos). Para usar el sistema sin restricciones:

1
Andá a Sistema → Licencia
2
Copiá tu ID de Instalación (único por cada computadora).
3
Contactá al desarrollador por WhatsApp o email para adquirir la licencia.
4
Ingresá el código recibido en el campo de activación.

🔫 Lector de Código de Barras

El sistema es completamente compatible con lectores de barras USB tipo HID (los más comunes).

Configuración

No requiere ninguna configuración. El lector funciona como un teclado: al escanear, el código se ingresa automáticamente en el campo activo.

Uso en Punto de Venta
  1. Asegurate que el cursor esté en el campo de búsqueda (se enfoca automáticamente al abrir la pantalla).
  2. Escaneá el código de barras del producto.
  3. El sistema lo busca automáticamente y muestra el producto.
  4. Confirmás la cantidad y se agrega al carrito.
Uso en Carga de Productos

Al crear o editar un producto, podés escanear el código de barras directamente en el campo «Código de Barras».

💡 Compatible con lectores Datalogic, Zebra, Honeywell, Symbol, y cualquier lector USB HID estándar.

💡 Tips y Trucos

  • Presioná Enter en el campo de búsqueda del POS para agregar el primer resultado al carrito.
  • Hacé backup del archivo almacen.db periódicamente — ahí está toda tu información.
  • Usá el módulo de Análisis para identificar qué productos prescindibles estás comprando y reducir gastos.
  • Configurá el stock mínimo de cada producto según tu rotación para recibir alertas a tiempo.
  • Las ventas en «Cuenta Corriente» solo están disponibles si el cliente está cargado en el sistema.
  • Desde Estadísticas, cambiá el año para comparar el rendimiento anual.
  • El PDF del catálogo es útil para tener una lista de precios impresa actualizada.
